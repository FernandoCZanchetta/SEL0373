O Sharp_Probe é um projeto de graduação da Escola de Engenharia de São Carlos da disciplina SEL0373 - Projetos em Internet das Coisas Este projeto tem o obejtivo de demonstrar o funcionamento de uma sonda climática, apresentando dados como posição, altura e temperatura.
